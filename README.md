# Projeto Organiza Festa

### O programa tem como objetivo organizar orçamento, cronogramas e tarefas para qualquer tipo de evento.

## Como utilizar a ferramenta:

1. Salve o arquivo
2. Rode o arquivo main.py

Quando o terminal abrir você verá:

```
=== SISTEMA ORGANIZA FESTA ===
1. Acessar Painel de Eventos
2. Sair
```
## Painel de Eventos

Aqui será possível que você veja todos os eventos cadastrados assim como seu:

* Índice
* Nome
* Data
* Tempo restante 

E as seguintes opções:

```
[N]ovo Evento   [G]erenciar Detalhes   [E]ditar   [R]emover   [V]oltar
```

### Ao criar novo evento:

Informe:

* Nome
* Tipo do evento
* Data
* Local
* Orçamento

### Ao gerenciar detalhes:

Você verá:

* Dias restantes
* Orçamento total
* Saldo disponível
* Lista de tarefas

### Ao editar evento: 

Você pode alterar:

* Nome
* Tipo do evento
* Data
* Local
* Orçamento

### Ao remover evento:

Remove o evento selecionado e suas respectivas tarefas.

## Limitações do programa:

* Interface por terminal
* Banco de dados limitado por conta do TXT
* Caso crie um HTML, deve ser deletado manualmente
